# Deploy NBA Analytics App to Streamlit Cloud

## Quick Deploy Steps

1. **Fork or push the repository to your GitHub account**
   - The NBA-Analytics-Streamlit-App-with-LangChain-Agent is already in your repo as a submodule

2. **Go to Streamlit Cloud**
   - Visit: https://share.streamlit.io/
   - Sign in with your GitHub account

3. **Deploy the app**
   - Click "New app"
   - Select repository: `petermartens98/NBA-Analytics-Streamlit-App-with-LangChain-Agent`
   - Main file path: `streamlit_app.py`
   - Click "Deploy"

4. **Access from iPhone**
   - You'll get a public URL like: `https://yourapp.streamlit.app`
   - Open it on any device including your iPhone!

## Benefits
- ✅ Free hosting
- ✅ Always accessible
- ✅ Automatic updates when you push to GitHub
- ✅ Works from anywhere (iPhone, Android, desktop)

## Note
For the Chat feature, you'll need to add your OpenAI API key in Streamlit Cloud secrets.
